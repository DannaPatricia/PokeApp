## Instalación y ejecución del proyecto

1. Instalación de dependencias

```sh
npm install
npm install axios
```

2. Ejecutar el proyecto

```sh
npm run dev
```

3. Compile and Minify for Production

```sh
npm run build
```

4. Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
